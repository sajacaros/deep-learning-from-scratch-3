## 1.3 [보충]넘파이의 다차원 배열
* 스칼라
  * 0차원 배열
* 벡터
  * 1차원 배열
* 행렬
  * 2차원 배열
* 텐서
  * 다차원 배열
  * 스칼라 = 0차원 텐서
  * 벡터 = 1차원 텐서
  * 행렬 = 2차원 텐서
``` 
x = np.array(1)
x.ndim # 0

x = np.array([1,2])
x.ndim # 1

x = np.array([[1,2],[3,4]])
x.ndim # 2
```
